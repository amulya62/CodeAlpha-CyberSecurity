Login System with Password Strength Checker 🔐

This is a simple Python project that allows users to register and login.
It also checks the strength of the password during registration.
Features
User registration
Password strength checking (Weak, Medium, Strong)
User login system
Stores user data in a text file
Technologies Used
Python

Regular Expressions 

How to Run
python login_system.py

Menu Options

Register a new user
Login with username and password
Exit the program
